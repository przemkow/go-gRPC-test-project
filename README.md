# Golang gRPC test project

## How to start:
### 1. build executable:
```
  make build
```

### 2. run server app:
```
  make run_server
```

### 3. run client app:
```
  make run_client
```

### Features:
  - [x] Create new todo 
  - [x] List all created todos
  - [ ] Update existing todo
  - [ ] Connect to DB
